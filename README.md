# Calificador de Profesores – Backend

Sistema backend desarrollado para permitir la calificación de docentes universitarios de forma controlada y confiable.

## Funcionalidades principales

- Registro y gestión de calificaciones de docentes.
- Restricción de calificación con un mínimo de 3 días entre evaluaciones del mismo usuario a un mismo profesor.
- Cálculo automático de promedios de calificación por docente.
- Asociación de docentes con materias.
- Perfil informativo de docentes (modo solo lectura).
- Validación de datos de entrada y control de errores.
- Exposición de una API REST documentada.

Proyecto orientado a la aplicación de reglas de negocio, modelado relacional y buenas prácticas en desarrollo backend.
